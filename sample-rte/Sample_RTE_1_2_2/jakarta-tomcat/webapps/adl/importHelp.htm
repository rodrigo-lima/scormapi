Course Import Help
Course Import allows an administrator to import a SCORM® Conformant course into the Sample Run-Time Environment. Courses conformant to The SCORM® Content Aggregation Model in SCORM® 1.2 will be accepted into the Sample RTE.

Import Instructions
  1. Enter the name that you would like to be given to the course that you are about to import.
  2. Click the Browse button and select the Zip file that contains the course. This Zip file MUST BE a valid SCORM Version 1.2 Application Profiles Content Aggregation Package.
  3. Choose the type of controls that the RTE will use while running the course. The current options are:
    • Flow - 'Previous' and 'Next' Buttons are used to navigate through the course.
    • Choice - A table of contents is presented that can be used to navigate through the course.
