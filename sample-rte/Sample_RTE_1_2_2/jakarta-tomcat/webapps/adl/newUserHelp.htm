Add User Account Help
Add User Account enables an administrator to add a user's information to the Sample Run-Time Environment.

Add User Account Instructions
  1. Enter the User ID.
  2. Enter the First Name.
  3. Enter the Last Name.
  4. Enter the Password.
  5. Confirm the Password.
  6. Select whether the user is to be an admin or not.
  7. Click the Submit button.
